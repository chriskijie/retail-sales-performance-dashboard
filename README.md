# Retail Sales Performance Dashboard
### Student Industrial Training Project – ARVYS Technologies

## Project Overview

The **Retail Sales Performance Dashboard** is an interactive Microsoft Excel dashboard developed as part of an **Industrial Training Programme at ARVYS Technologies**. The project demonstrates the complete data analysis workflow—from cleaning and validating raw business data to generating actionable business insights through Pivot Tables, Pivot Charts, KPIs, and interactive dashboards.

The objective of this project was to simulate a real-world business intelligence solution capable of helping management monitor sales performance, customer behaviour, product performance, and operational efficiency.

---

## Project Objectives

- Clean and prepare messy retail sales data.
- Validate data accuracy and consistency.
- Perform business-focused exploratory analysis.
- Build Pivot Tables for summarising business metrics.
- Design an interactive dashboard using slicers and KPIs.
- Communicate insights through effective data visualisation.

## Data Cleaning Activities

The following data preparation steps were carried out:

- Removed duplicate records.
- Standardised text formatting.
- Corrected inconsistent date formats.
- Validated missing values.
- Reviewed negative transaction values.
- Treated invalid customer ratings.
- Standardised product categories.
- Ensured data type consistency.
- Applied business validation rules.
